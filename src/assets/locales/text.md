# Конструктор сценариев для Ереси Хоруса
Сценарий для Ереси Хоруса состоит из пяти частей: тип расстановки, основные цели, побочные цели, длина и спецправила.
Система предлагает несколько вариантов определения сценария.
Во-первых, вы можете произвольно выбрать все параметры сценария.
Во-вторых, вы можете определить параметры броском кубика. Для определения типа расстановки, основной цели и длины, киньте один д6 и выберите соответствующий вариант. Для определения спецправил и побочных целей, кидайте д6 за каждое из спецправил; оно выбрано, если число на кубике больше значения, указанного в правила. Делайте так, пока не будет выбрано достаточное количество.

Миссии считаются core миссиями, и используются все правила из книги правил. Если спецправила миссии противоречат книге правил, приоритет имеют специальные правила миссии.

## Основные цели
В этом разделе определяется количество основных целей. Для расстановки целей используются правила из основной книги правил: цели не могут быть выставлены в зонах расстановки, непроходимом ландшафте, или ближе чем в 12 дюймах от других целей. Если в миссии нечетное количество целей, перед выставлением целей поставьте одну из них в центр стола. Если на столе нет места, куда можно поставить цель, допускается ставить ее внутри зоны расстановки. 

### 1: 2 цели
Каждая цель приносит 3 победных очка.
### 2: 3 цели
Каждая цель приносит 2 победных очка. Центральная цель приносит 4 победных очка.
### 3: 4 цели
Каждая цель приносит 1 победное очко.
### 4: 5 целей
Каждая цель приносит 1 победное очко.  Центральная цель приносит 3 победных очка.
### 5: 6 целей
Каждая цель приносит 1 победное очко.
### 6: 7 целей
Каждая цель приносит 1 победное очко. Центральная цель приносит 2 победных очка.
